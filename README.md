# LoadingScreen-MTA-SA
A simple loading screen that displays scenarios and includes 3 online radio stations so that downloading resources is not annoying for customers.

## Includes:

* 3 online radio stations with text displaying the current station.
* Volume fade-out at the beginning and end of songs.
* More than 5 customizable backgrounds.
* Classic design with native MTA features; no HTML.
